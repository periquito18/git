Examen Git y Github

Alumn@: SMP

  1. ¿Que es un sistema de control de versiones?

    Un sistema de control de versiones o "control de código fuente" es la práctica de rastrear y gestionar los cambios en el código del software

  2. ¿Cual es la diferencia entre git y github?

    Git es la herramienta para realizar el seguimiento de los cambios. Github es el lugar para almacenar los proyectos y trabajar en equipo

  3. ¿Cómo creo una rama en git?

    Entras en Github, entras en un reposiorio, buscas la opción branch y seleccionas la opción "new branch". En línea de comandos, escribimos "git branch nombre_rama"

  4. ¿Cómo clono un repositorio?

    Entras en un repositorio de Github, pulsas sobre la opcción "clone" y copiamos el enlace ssh. Luego en línea de comandos, escribimos "git clone enlace_ssh"
